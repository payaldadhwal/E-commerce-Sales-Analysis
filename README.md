# 📊 E-Commerce Sales Dashboard (Excel Project)

## 📌 Project Overview

This project presents an interactive **E-Commerce Sales Dashboard** built using Microsoft Excel. The goal of this project is to analyze raw sales data and convert it into meaningful insights using data cleaning, pivot tables, and visual dashboards.

The dashboard helps in understanding sales performance, customer behavior, order trends, and channel-wise distribution.


## 🎯 Objectives

* Clean and prepare raw e-commerce data
* Perform data analysis using Pivot Tables
* Create an interactive and visually appealing dashboard
* Extract actionable insights for business decision-making


## 🛠️ Tools & Techniques Used

* Microsoft Excel
* Data Cleaning
* Pivot Tables
* Pivot Charts
* Data Visualization
* Slicers & Filters


## 🔄 Project Workflow

### 1. Data Cleaning

* Removed duplicates and handled missing values
* Standardized column names and formats
* Ensured consistency in categories (e.g., gender, age group, states)

### 2. Data Processing

* Created calculated columns where necessary
* Structured data for analysis

### 3. Data Analysis (Pivot Tables)

* Built multiple pivot tables to analyze:

  * Sales vs Orders trend (monthly)
  * Sales by state
  * Orders by channels
  * Age group vs gender analysis
  * Order status distribution

### 4. Dashboard Creation

* Designed an interactive dashboard using:

  * Charts (bar, pie, line)
  * Slicers (Month, Category, Channel)
* Enabled dynamic filtering for better user experience


## 📊 Dashboard Insights

* Majority of orders are **successfully delivered (~90%+)**
* **Amazon & Flipkart** contribute the highest sales share
* **Women customers (~64%)** dominate purchases
* Top performing states include:

  * Maharashtra
  * Karnataka
  * Uttar Pradesh
* Adult age group contributes the highest number of orders
